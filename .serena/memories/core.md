# ArchGuard Core

ArchGuard is a local-first CLI tool and GitHub Action for detecting architectural drift using LLMs and embeddings.
It compares code changes against Architectural Decision Records (ADRs) stored in markdown files with strict YAML frontmatter. It is distributed via cross-platform Go binaries and the GitHub Marketplace.
